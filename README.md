# BazarSide — Coming Soon Landing

Simple static landing and contact pages for BazarSide. Open `index.html` in a browser to view.

Files added:
- index.html — main coming-soon page
- contact.html — contact form (opens mail client)
- assets/css/styles.css — styles
- assets/js/main.js — small JS for forms

No build required — just open `index.html` or serve the folder with a static server.

Example local preview using Python 3:
```bash
python -m http.server 8000
# then open http://localhost:8000
```
